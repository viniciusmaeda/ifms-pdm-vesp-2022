Comandos
cd -> navegar entre as pastas
cls/clear -> limpar a tela
dir -> listar o conteúdo de uma pasta
mkdir -> criar uma pasta em branco
rm -> remover uma pasta vazia
code . -> abrir o VS Code na pasta do projeto
